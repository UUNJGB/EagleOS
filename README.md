# EagleOS

EagleOS Based on the Programming Language Python! EagleOS Was Created for one purpose and that purpose was to act like a Real OS!

## Instalation

1. Go to https://github.com/UUNJGB/EagleOS/
2. Go to the Release Pages
3. Download the Latest one
4. Extract it
5. Run "run.bat" in EagleOSGUI
